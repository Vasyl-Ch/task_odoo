## Запуск

```bash
docker compose up -d
```

Дочекайся поки контейнери запустяться (10–15 секунд), потім відкрий браузер.

## Доступ


| Веб-інтерфейс Odoo | http://localhost:8069 |
| REST API (список книг) | http://localhost:8069/library/books |

При першому запуску потрібно створити базу даних через форму на головній сторінці.

## Встановлення модуля

1. Увійди в систему
2. Перейди в **Apps**
3. Натисни **Update App List**
4. Знайди **Library Management** → **Install**

## Що реалізовано і навіщо

### Модель `library.book`
Зберігає книги бібліотеки. Поле `is_available` показує чи можна видати книгу. 
Автоматично змінюється при видачі та поверненні.

### Модель `library.rent`
Зберігає записи оренди — хто і коли взяв книгу, коли повернув.
Містить перевірку: одну книгу не можна видати двічі, якщо вона ще не повернена.

### Wizard видачі книги
Відкривається кнопкою **"Видати книгу"** на формі книги. 
Дозволяє вибрати читача і автоматично створює запис оренди.

### Кнопка повернення
На формі оренди є кнопка **"Повернути книгу"** — встановлює дату повернення і робить книгу знову доступною.

### REST API
```
GET http://localhost:8069/library/books
```
Повертає JSON зі списком усіх книг та їх статусом. Доступний без авторизації.
